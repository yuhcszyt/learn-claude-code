# 学习笔记

[[index|返回学习笔记索引]]

这里存放按项目持续维护的 GitHub AI 学习笔记。

命名建议：

```text
project-name.md
```

例如：

```text
langgraph.md
moneyprinterturbo.md
```

新建笔记时，从 [[templates/project-study|项目学习笔记模板]] 复制，再把索引补到 [[index|学习笔记索引]]。后续再次学习同一项目时，继续维护同一个文件。
