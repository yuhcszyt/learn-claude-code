# LangChain Workspace

This folder is reserved for Python LangChain and LangGraph learning experiments.
Java is still useful for backend integration, but this folder's main line is Python.
Spring AI and LangChain4j are reference-only unless a concrete Java LLM project needs them.

Suggested future layout:

- `notes/` - learning notes and concept maps.
- `examples/python-agent-demo/` - small Python LangChain / LangGraph agent examples.
- `projects/rag-notes/` - larger RAG experiments over this repository's notes.
- `data/` - local input data, ignored by default when needed.
